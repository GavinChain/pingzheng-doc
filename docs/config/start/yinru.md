# 依赖关系

- H5(JSP,PHP)：

```
<script type="module" crossorigin>
  import pingzheng from '../../../boozjs/2.0.0/lib/es/pingzheng.js
</script>
```

- Vue3：

```
import pingzheng from 'pingzheng/pingzheng'
```

- Vue2：

```
import pingzheng from 'pingzheng/pingzheng'
```

- 例子：

```
import pingzheng from 'pingzheng/pingzheng'
pingzheng.openNewPingZheng()
```
